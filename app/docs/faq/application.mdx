Frequently asked questions that are app specific.

#### _How can I install Mochi?_

Mochi is currently available on TestFlight. It is still a work-in-progress so expect
things to break as new versions are introduced.

#### _Are there any modules preinstalled?_

No, there are no modules preinstalled when dowloading Mochi.

#### _I am having issues with X module, can you fix it?_

No, I am in no control of any issues related to external modules other than issues caused by the app itself.

#### _Can I contribute to this app?_

Yes! Any contribution is greatly appreciated! Go [here](/docs/contribution) for more information on how you can contribute.
